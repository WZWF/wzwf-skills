# 能力一：代码传记（Git 考古）

追溯文件/类的完整 Git 演化历史，生成叙事性时间线。

## 输入

- 文件路径，或 Java 类名

## 分析步骤

1. 获取完整 commit 列表（`--follow` 追踪重命名）
2. 定位首次提交，读取初始版本 diff，推断初始目的
3. 按时间排序，标记重大重构 / feat / fix 节点
4. 统计贡献者：`git shortlog -sn -- <file>`
5. 可选：对比首版与当前版，总结架构漂移

> Git 命令详见 [git-commands.md](git-commands.md)。

<这段代码是什么、为什么存在、经历过什么>

## 时间线

...
