# 技术栈参考与注意事项

## 常见技术栈出题指引

针对用户常见栈，优先深挖以下方向（**仅在代码中存在时才出**）：

| 技术 | 深挖方向 |
|------|----------|
| 分布式锁 | 加锁/解锁原子性、锁续期、可重入、防误删、Redisson vs 自研 |
| Redis + Lua | 脚本参数、EVALSHA、失败回退、与 MULTI/EXEC 对比 |
| AOP | 切面顺序、@Around 异常处理、代理类型（JDK/CGLIB） |
| SpEL | Key 表达式解析、参数绑定、表达式注入风险 |
| Spring Boot Starter | 自动配置条件、配置属性绑定、Bean 加载顺序 |

## 注意事项

- 用户指定子目录时，只分析该目录，不臆测其他模块
- 代码量过大时，优先 P0/P1 文件，告知用户可扩展分析范围
- 模拟面试模式：逐题提问，等用户回答后再给参考答案和追问
